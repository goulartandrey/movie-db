## Funcionalidades
  - Busca de filmes por título.
  - Listagem de filmes populares e lançamentos recentes.

## Tecnologias Utilizadas
  - React
  - Javascript
  - Vite

## Uso
  - Clone o repositorio git clone https://github.com/goulartandrey/movie-db.git
  - cd movie-db
  - npm install
  - npm run dev
  - Acesse localhost:5173
